# A Lush Theme based on Ayu Dark

See: [lush](http://git.io/lush.nvim) for more information on Lush and a helper script
to setup your repo clone.

## Supported plugins

- barbar
- bufferline
- cmp
- diffview
- dropbar
- flash
- floaterm
- gitsigns
- hop
- illuminate
- indent_blankline
- lspconfig
- lspsaga
- lualine
- navbuddy
- neotree
- noice
- notify
- nvim-dap-ui
- nvim-treesitter-context
- satellite
- scrollbar
- semshi
- telescope
- todo-trouble
- treesitter
